# 堆叠图和输入方式改进说明

## 修改日期
2025-11-21

## 修改内容

### 1. 堆叠图按压力排序
- **功能**: 自动从文件名中提取压力值，按从小到大排序
- **实现**: 新增 `_extract_pressure_from_filename()` 方法
- **支持格式**:
  - `sample_25.5GPa.xy`
  - `data_10_GPa.xy`
  - `test_15.xy` (数字会被识别为压力)

### 2. 压力标签显示
- **位置**: 每个堆叠曲线的左侧
- **格式**: `X.X GPa` (保留1位小数)
- **样式**: 白色背景框，灰色边框，便于阅读

### 3. 横坐标范围优化
- **最小值**: 向上取整 (`np.ceil`)
- **最大值**: 向下取整 (`np.floor`)
- **优势**: 提供整齐的坐标范围，便于读数

### 4. 输入方式改变
- **原来**: "Input Folder (containing .h5 files)" + Browse Folder 按钮
- **现在**: "Input .h5 File" + Browse 按钮
- **行为**:
  - 用户选择**任意一个** .h5 文件
  - 程序自动处理该文件所在文件夹下的**所有** .h5 文件
  - 支持向后兼容：如果输入是文件夹路径，仍然可以正常工作

## 使用示例

### 输入文件示例
```
/data/experiment/
  ├── sample_10.5GPa.h5
  ├── sample_15.2GPa.h5
  ├── sample_20.8GPa.h5
  └── sample_25.0GPa.h5
```

### 操作步骤
1. 点击 "Browse" 按钮
2. 选择 **任意一个** .h5 文件（例如 `sample_15.2GPa.h5`）
3. 程序会自动处理整个文件夹下的所有 .h5 文件
4. 生成的堆叠图会按压力排序：10.5 → 15.2 → 20.8 → 25.0 GPa

### 生成的堆叠图特点
- ✅ 按压力从小到大排序
- ✅ 左侧显示压力标签
- ✅ 横坐标范围整齐（取整）
- ✅ 更大的图像尺寸（12x10英寸）
- ✅ 日志中显示压力范围信息

## 技术细节

### 压力提取正则表达式
```python
r'(\d+\.?\d*)\s*(?:GPa|gpa|_GPa|_gpa)?'
```
- 匹配整数或小数
- 可选的 GPa 标识（大小写不敏感）
- 支持下划线分隔

### 排序算法
```python
file_pressure_pairs.sort(key=lambda x: x[1])
```
按压力值（元组第二项）升序排序

### 压力标签位置计算
```python
ax.text(x_min - 0.02 * (x_max - x_min),
       i * offset_value + np.mean(y[:10]),
       f'{pressure:.1f} GPa', ...)
```
- X位置：稍微偏离左边界（2%）
- Y位置：曲线的垂直位置 + 初始几个点的平均高度

## 向后兼容性
- ✅ 仍支持直接输入文件夹路径
- ✅ 如果文件名中没有压力信息，默认为 0.0 GPa
- ✅ 保持原有的所有功能不变

## 已知限制
- 压力提取依赖于文件名格式，建议使用标准命名：`xxx_压力GPa.h5`
- 如果多个文件压力值相同，会保持文件名的字母顺序
- 压力标签位置固定在左侧，不会动态调整避免重叠
