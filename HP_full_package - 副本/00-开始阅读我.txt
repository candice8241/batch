🚀 快速开始 - 三步生成EXE
══════════════════════════════════════════════════════════════════════════

方式1️⃣：【最简单】一键打包（强烈推荐新手）
──────────────────────────────────────────────────────────

Windows用户：
  ➊ 双击运行 → 快速打包.bat
  ➋ 等待完成（约5-10分钟）
  ➌ 在 dist\XRD_PostProcessing 找到 .exe 文件

Linux/Mac用户：
  ➊ 打开终端，运行 → ./快速打包.sh
  ➋ 等待完成（约5-10分钟）
  ➌ 在 dist/XRD_PostProcessing 找到可执行文件


方式2️⃣：【标准】使用现有脚本
──────────────────────────────────────────────────────────

Windows用户：
  双击运行 → build.bat

Linux/Mac用户：
  运行 → ./build.sh


方式3️⃣：【手动】命令行打包（适合高级用户）
──────────────────────────────────────────────────────────

步骤1：安装依赖
  pip install -r requirements_gui.txt

步骤2：执行打包
  pyinstaller --clean xrd_app.spec

步骤3：查看结果
  Windows: dist\XRD_PostProcessing\XRD_PostProcessing.exe
  Linux/Mac: dist/XRD_PostProcessing/XRD_PostProcessing

💡 重要提示
══════════════════════════════════════════════════════════════════════════

1. 打包前要求：
   ✓ Python 3.8 或更高版本
   ✓ 可以联网（用于下载依赖包）
   ✓ 约2GB可用磁盘空间

2. 打包时间：
   ⏱ 首次打包：5-10分钟（需下载依赖）
   ⏱ 再次打包：2-5分钟

3. 生成的文件：
   📁 整个 dist/XRD_PostProcessing 文件夹（约200-500MB）
   📦 可以复制到任何电脑直接运行（无需安装Python）

4. 推荐方式：
   🌟 新手：使用"快速打包.bat"或"快速打包.sh"
   🌟 有经验：使用"build.bat"或"build.sh"
   🌟 高级用户：直接使用命令 pyinstaller --clean xrd_app.spec
